# 作用域

## 概念

作用域是定义变量的可访问范围，它规定了变量的可见性和声明周期。在程序执行时，JS 引擎会根据变量声明的位置来确定其作用域。

## 分类

### 全局作用域

1. 全局变量是挂载在 window 对象上的变量，拥有全局作用域，即在任何地方都能访问
2. 在 JS 中，任何没有定义而直接被赋值的变量默认是一个全局变量
3. 全局变量可能会引起命名冲突问题

### 函数作用域

1. 在函数中定义的变量只能在函数内部访问，称为局部变量，拥有函数作用域
2. 当函数执行完毕后，局部变量会被销毁

### 块级作用域

1. ES6 新增了块级作用域，由 `let` 和 `const` 定义的变量只能在块级作用域中被访问
2. 存在“暂时性死区”，即在代码块中，在使用 `let` 和 `const` 声明变量之前，变量不可用

## 作用域链

1. 作用域之间可以存在嵌套的关系，也就是所谓的作用域链
2. 在访问变量时，JS 引擎会由内向外搜索变量
