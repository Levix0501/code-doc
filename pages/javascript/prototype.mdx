# 原型和原型链

## 原型

JS 中是使用构造函数来创建对象的，构造函数本身也是一个对象，它的`prototype`属性指向一个对象，这个对象包含了可以由该构造函数的实例共享的属性和方法。

由构造函数创建的实例对象存在一个名为`__proto__`的属性，指向其构造函数的`prototype`属性指向的对象，该对象就称为实例的原型
。

## 原型链

当访问一个对象的属性时，如果该对象内部不存在此属性，JS 引擎就会去它的原型对象上找这个属性，如果找不到，就会去原型对象的原型对象上寻找，如此形成了所谓的原型链。原型链的终点时是`Object.prototype`，该对象的`__proto__`指向`null`。
