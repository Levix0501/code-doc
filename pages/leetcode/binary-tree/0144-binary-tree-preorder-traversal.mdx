# 二叉树的前序遍历

[题目链接](https://leetcode.cn/problems/binary-tree-preorder-traversal/description/)

## 题目大意

输出二叉树的前序遍历

## 思路与代码

### 方法一：递归

按照`根-左-右`顺序递归遍历即可

```js
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var preorderTraversal = function (root) {
  const ret = [];

  const preorder = (node) => {
    if (!node) return null;
    ret.push(node.val);
    preorder(node.left);
    preorder(node.right);
  };

  preorder(root);

  return ret;
};
```

- 空间复杂度：O(n)
- 时间复杂度：O(n)

### 方法二：迭代

借助栈进行迭代

```js
/**
 * Definition for a binary tree node.
 * function TreeNode(val, left, right) {
 *     this.val = (val===undefined ? 0 : val)
 *     this.left = (left===undefined ? null : left)
 *     this.right = (right===undefined ? null : right)
 * }
 */
/**
 * @param {TreeNode} root
 * @return {number[]}
 */
var preorderTraversal = function (root) {
  const ret = [];
  const stack = [];
  let node = root;
  while (node || stack.length > 0) {
    while (node) {
      ret.push(node.val);
      stack.push(node);
      node = node.left;
    }
    node = stack.pop();
    node = node.right;
  }
  return ret;
};
```

- 空间复杂度：O(n)
- 时间复杂度：O(n)
