# 变量与常量

变量的值不能更改可能让你想起其他另一个很多语言都有的编程概念：`常量(constant)`。

与[不可变变量](/rust/lang/variable#不可变变量)一样，常量也是绑定到一个常量名且不允许更改的值。

Rust 常量的命名约定是全部字母都使用大写，并使用下划线分隔单词

```rs
const MAX_POINTS: u32 = 100_000;
```

常量使用 const 关键字而不是 let 关键字来声明，并且值的类型必须标注

常量在声明时必须赋值

```rs
const a: i32 = 1;
const b: i32; // ❌错误

let c;
c = 1;
```
