# 基本数学运算

Rust 支持所有数字类型的基本数学运算：加法、减法、乘法、除法和取模运算，数字类型包括[整数类型](/rust/lang/scalar-types/integer)和[浮点类型](/rust/lang/scalar-types/floating-point)

```rs
fn main() {
    // 加法
    let sum = 5 + 10;

    // 减法
    let difference = 95.5 - 4.3;

    // 乘法
    let product = 4 * 30;

    // 除法
    let quotient = 56.7 / 32.2;

    // 求余
    let remainder = 43 % 5;
}
```
