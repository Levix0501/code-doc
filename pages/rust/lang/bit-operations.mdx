# 位运算

Rust 支持对两个整数类型的数进行位运算，位运算符如下：

| 运算符    | 说明                                                     |
| --------- | -------------------------------------------------------- |
| & 位与    | 相同位置均为 1 时则为 1，否则为 0                        |
| \| 位或   | 相同位置只要有 1 时则为 1，否则为 0                      |
| ^ 异或    | 相同位置不相同则为 1，相同则为 0                         |
| ! 位非    | 把位中的 0 和 1 相互取反，即 0 置为 1，1 置为 0          |
| \<\< 左移 | 所有位向左移动指定位数，右位补 0                         |
| >> 右移   | 所有位向右移动指定位数，带符号移动（正数补 0，负数补 1） |

```rs
fn main() {
    let a: i32 = 2;
    let b: i32 = 3;
    let c = a & b;
    let d = a | b;
}
```
